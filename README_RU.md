## О проекте

**Графиня — Панель управления** — это полнофункциональное Next.js приложение для работы с платформой мониторинга и визуализации данных **Графиня** от Лаборатории Числитель. Панель предоставляет интуитивный интерфейс для управления дашбордами, исследований данных, настройки источников данных и мониторинга инфраструктуры.

Проект разработан как самостоятельный фронтенд, взаимодействующий с бэкендом Графини через REST API. Панель включает **8 основных видов**, систему командной палитры, демо-режим для работы без сервера, режим презентации для NOC/SOC-дисплеев, шаблоны дашбордов, массовые действия и **8 цветовых палитр**.

Полные описания доступны на русском и английском языках:
- [README на русском языке](README_RU.md)
- [README in English](README_EN.md)

### Ключевые возможности

- **8 основных видов** — дашборды, explorer, источники данных, плагины, модули, конструктор, активность, настройки
- **Дашборды** — список и детальный просмотр с виджетами (Line, Bar, Pie, Table, Area, Gauge), drag-and-drop переупорядочивание
- **Explorer** — редактор запросов, история, сохранённые запросы, графики/таблицы/JSON
- **Командная палитра** (Ctrl+K) — глобальный поиск и навигация
- **Демо-режим** — полностью функциональная работа без подключения к серверу
- **Режим презентации** — полноэкранный режим для NOC/SOC-дисплеев с авто-ротацией и авто-обновлением
- **Шаблоны дашбордов** — 9 готовых шаблонов в 4 категориях
- **8 цветовых палитр** — Янтарь, Океан, Закат, Лес, Монохром, Контраст, Пастель, Светофор
- **Массовые действия** — пакетный экспорт, дублирование, тегирование и удаление
- **История изменений** — авто-снимки состояния дашборда с восстановлением версий
- **Метрики запросов** — отслеживание времени выполнения, размера ответа и кэша
- **Мониторинг системы** — live-статус 12 сервисов с CPU, памятью, диском, сетью

### Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/dupleymi-aup/grafinya-panel.git
cd grafinya-panel

# Установить зависимости
bun install

# Запустить в режиме разработки
bun run dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)
